# 画布格式规范

> SVG 基础规则请参阅 shared-standards.md。

## 格式速查表

| 格式 | viewBox | 比例 | 使用场景 |
|--------|---------|-------|----------|
| PPT 16:9 | `0 0 1280 720` | 16:9 | 商务演示、会议汇报 |
| PPT 4:3 | `0 0 1024 768` | 4:3 | 传统投影仪、学术演讲 |
| 小红书 | `0 0 1242 1660` | 3:4 | 图文分享、知识帖 |
| 微信朋友圈 / IG | `0 0 1080 1080` | 1:1 | 方形海报、品牌展示 |
| Story / 抖音 | `0 0 1080 1920` | 9:16 | 竖版故事、短视频封面 |
| 微信公众号头图 | `0 0 900 383` | 2.35:1 | 公众号文章封面 |
| 横版 Banner | `0 0 1920 1080` | 16:9 | 网页横幅、数字大屏 |
| 竖版海报 | `0 0 1080 1920` | 9:16 | 手机屏、电梯广告 |
| A4 印刷 | `0 0 1240 1754` | 1:sqrt(2) | 印刷海报、传单 |

## 布局原则

### 横版（16:9、4:3、2.35:1）
- 视觉动线：Z 字型，从左到右
- 边距：40-80px
- 布局方式：多列、左右分栏、网格
- 卡片尺寸（16:9）：单行高 530-600px，双行高 265-295px

### 竖版（3:4、9:16）
- 视觉动线：从上到下
- 边距：60-120px
- 布局方式：单列、上下分栏、卡片堆叠
- 卡片尺寸（3:4）：高度 400-600px，间距 40-60px

### 方形（1:1）
- 视觉动线：中心向外辐射
- 边距：60-100px
- 核心区域：约 800x800px

## 各格式设计要点

| 格式 | 标题区域 | 内容区域 | 特殊说明 |
|--------|-----------|--------------|---------------|
| PPT | 80-100px | 充分利用全宽 | 页码置于右下角 |
| 小红书 | 180-240px（加粗） | 上下留白要充足 | 底部品牌区 120-160px |
| 微信朋友圈 | 200-280px | 居中 500-600px | 底部二维码区 150-200px |
| Story | — | 中间 1500px | 顶部安全区 120px，底部 180px |
| 微信公众号头图 | 居中/左对齐 48-72px | — | 图片放右侧或作为背景 |

## viewBox 示例

```xml
<svg width="1280" height="720" viewBox="0 0 1280 720">   <!-- PPT 16:9 -->
<svg width="1242" height="1660" viewBox="0 0 1242 1660"> <!-- 小红书 -->
<svg width="1080" height="1080" viewBox="0 0 1080 1080"> <!-- 微信朋友圈 -->
<svg width="1080" height="1920" viewBox="0 0 1080 1920"> <!-- Story -->
<svg width="900" height="383" viewBox="0 0 900 383">     <!-- 微信公众号头图 -->
```
