# Design Systems 索引

> 72套品牌 Design System Token 速查表。当用户提到具体品牌时，从此索引定位对应 token 文件加载。

## 使用方式

1. 用户提到品牌名 → 在此表查找 → `read references/design-systems/<brand>.md`
2. 将 token 色板注入 `design_spec.md` 的配色方案（使用 Token 引用语法，见 `token-reference.md`）
3. 将字体栈注入字体方案
4. 参考间距/圆角风格调整布局
