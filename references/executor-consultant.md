## 角色定义

数据驱动的咨询风格 SVG 设计执行器。适用于商业分析报告、市场调研、运营复盘、战略建议等**通用咨询**场景。强调结构化信息呈现和数据可视化；风格简洁、清晰、专业。

## 咨询风格专属数据可视化技巧

### 1. KPI 仪表盘设计

KPI 卡片是咨询报告中最常见的元素。标准布局（1280x720）：

```
4 卡片布局：每张卡片 280x180，间距 30
  卡片 1：x=45,  y=160
  卡片 2：x=355, y=160
  卡片 3：x=665, y=160
  卡片 4：x=975, y=160
```

**卡片内部结构**（从上到下）：

| 区域 | 内容 | 字号 | 样式 |
|------|------|------|------|
| 图标行 | data-icon 图标 | 32x32 | 主题色 |
| 指标名称 | "月活跃用户" | 14px | 灰色 #64748B |
| 核心数字 | "120万" | 36-42px | 加粗，深色 |
| 趋势标注 | "+12.3% vs 上月" | 12px | 绿色=上升 / 红色=下降 |

**趋势箭头约定**：
- 上升：`arrow-trend-up` 图标 + 绿色文字
- 下降：`arrow-trend-down` 图标 + 红色文字
- 持平：水平线图标 + 灰色文字

### 2. 图表配色约定

咨询风格图表使用**单色系深浅渐变**，而非彩虹色：

```
主系列：   主题色 100% 不透明度
对比系列：  主题色 60% 不透明度
基线：      #94A3B8（灰色虚线）
高亮：      强调色（仅用于关键数据点）
```

### 3. 数据标注原则

- **直接数据标签**：在柱状图顶部直接放置数值——无需图例
- **标注趋势线**：在关键拐点添加文字说明（"政策变更"、"产品上线"）
- **对比基线**：用灰色虚线标注行业均值/目标值
- **单位统一**：同一图表内保持相同的数字单位和精度

### 4. 表格设计

咨询报告中频繁使用表格呈现精确数据：

| 设计要素 | 规格 |
|----------|------|
| 表头 | 深色背景 + 白色文字，font-weight="bold" |
| 斑马纹 | 隔行使用 `fill-opacity="0.05"` 浅色背景 |
| 数字对齐 | 数字右对齐，文本左对齐 |
| 高亮行 | 关键行用强调色 `fill-opacity="0.1"` 标记 |
| 边框 | 仅用水平线分隔行；避免完整网格线 |

## 咨询风格专属布局模式

### MECE 分解树

主干在左，分支向右展开——互不重叠、完全穷尽：

```
主指标 ──┬── 分支 A (45%)
         ├── 分支 B (30%)
         ├── 分支 C (20%)
         └── 分支 D (5%)
```

- 使用 `<line>` 连接主干和分支
- 每个分支有 `<rect>` 容器 + 百分比标注
- 总和必须 = 100%（或明确标注"其他"）

### 驱动因素树

顶层指标 → 分解因素 → 行动。适合绩效归因分析：

```
营收增长 15%
  ├── 客单价 +8%   → 高端产品线扩展
  ├── 客户数 +5%   → 新渠道获客
  └── 复购率 +2%   → 会员计划优化
```

### 左图右文（图表解读）

核心咨询报告布局：图表在左，关键洞察在右：

```
图表区域：x=40, y=120, w=700, h=480
洞察区域：x=780, y=120, w=460, h=480
  - 核心结论（加粗，16px）
  - 3-5 个要点（14px）
  - 数据来源（12px，灰色）
```

## 专业表达规范

### 页面标题

咨询风格的页面标题应该是**断言式标题**，而不是描述性标题：

| 类型 | 描述型（避免） | 断言型（推荐） |
|------|---------------|---------------|
| 市场 | "市场概览" | "国内市场同比增长 23%，大幅超越全球均值" |
| 竞争 | "竞争分析" | "三大竞品在渠道覆盖上存在明显短板" |
| 财务 | "财务数据" | "毛利率连续四个季度改善，突破行业天花板" |

### 数据来源标注

每个包含数据的页面都必须在底部标注来源：

```xml
<text x="40" y="700" font-size="10" fill="#94A3B8">
  来源：国家统计局 2025 年度报告；内部团队分析
</text>
```

### 核心要点框（Takeaway Box）

咨询风格内容页应在标题下方包含 **Takeaway Box**：

```
位置：x=40, y=80, w=1200, h=50
背景：浅主题色（fill-opacity="0.08"）
文字：14-16px，一句话总结本页核心要点
```

## 演讲备注风格

### 叙述基调

咨询风格的演讲备注采用**结论驱动**方式——先说结论，再呈现支撑证据。专业、简洁、有说服力的语调。

### 舞台指示标记

| 标记 | 用途 | 示例 |
|------|------|------|
| `[Pause]` | 关键结论后留白，让听众消化 | "这意味着我们的市场份额正在被蚕食。[Pause]" |
| `[Data]` | 口语化转述数字 | "[Data] 85% → 超过十分之八的客户反映了这个问题" |
| `[Transition]` | 承接上页，必须放在每页文本开头 | "[Transition] 基于上述分析，我们来看具体建议。" |

### 备注写作指南

- **结论先行**：每页备注的第一句话就是核心要点
- **数字口语化**：30% → "大约三分之一"，85% → "超过十分之八"，2.5倍 → "两倍半"
- **证据紧跟**：结论之后立即给出 2-3 个支撑数据/事实
- **专业术语**：使用"洞察"、"驱动因素"、"关键杠杆"等术语
- **要点结构**：`要点：(1) 核心结论 (2) 支撑数据 (3) 下一步行动`

### 备注示例

```markdown
# 03_key_findings

[Transition] 基于市场扫描，我们提炼出三个关键发现。

第一，增长的主要驱动力正从获客转向留存。
[Data] 新客获取成本同比上升近 40%，而复购率的改善贡献了超过 60% 的营收增长。[Pause]

第二，下沉市场增速大幅超过一二线城市。
第三，25-35 岁女性用户群的 ARPU 是整体均值的 1.8 倍。

要点：(1) 增长引擎从获客转向留存 (2) 下沉市场机会窗口 (3) 高价值用户画像
时长：2 分钟
```

## 自检补充（咨询风格专属）

- [ ] 数据与源数据一致——不得编造
- [ ] 每个包含数据的页面底部都有来源标注
- [ ] 页面标题是断言型，而非描述型
- [ ] KPI 卡片有趋势箭头和对比标注
- [ ] 图表配色统一，使用单色系而非彩虹色
- [ ] 备注结论先行，数字口语化
