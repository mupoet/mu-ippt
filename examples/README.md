# mu-ippt Examples

This directory contains example PPTX files generated by mu-ippt, showcasing its capabilities.

## Design Philosophies Showcase

📁 [`design-philosophies/muippt-design-philosophies_202606.pptx`](./design-philosophies/muippt-design-philosophies_202606.pptx)

A 31-slide showcase demonstrating mu-ippt's 15 design philosophies across 6 major design lineages — each philosophy presented with a cover slide and a Visual DNA breakdown:

| Lineage | Design Philosophies |
|---------|-------------------|
| Information Architecture | Pentagram, Fathom, Müller-Brockmann, Tufte |
| Kinetic Poetics | Field.io, Sagmeister |
| Minimalism | Build, Notion, Apple |
| Experimental Avant-garde | Instrumental, Olschinsky |
| Eastern Philosophy | Takram, Kenya Hara, Fukasawa |

Each philosophy section includes:

- **Visual DNA** — Color palette, typography, layout grid, SVG conversion traits
- **Design Principles** — Core philosophy and constraints
- **Cover Slide** — Distinctive visual identity for the philosophy

The final slide provides a **selection guide** — how to choose the right philosophy based on audience type.

### Preview

> Download the `.pptx` file and open in PowerPoint, Keynote, WPS, or Google Slides to view the full presentation with all design details.
