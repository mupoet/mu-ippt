⚡ Features

Everything You Need 💪

Professional-grade capabilities, zero subscription, fully open source.

📊

119 Chart Types

From basic bar charts to Sankey diagrams, treemaps, waterfall charts, and KPI dashboards. All rendered as native vector shapes.

🏛️

40 Consulting Templates

McKinsey / BCG / Bain grade. SWOT, Porter's Five Forces, BCG Matrix, Value Chain, org charts, talent 9-box, and more.

🖼️

6,732 Vector Icons

All SVG format, lossless scaling. Covers tech, finance, medical, education, retail, and more. Built-in, no internet needed.

🔄

Document to PPT

Convert PDF, DOCX, Markdown, HTML, and EPUB into polished presentations. Preserves structure, auto-matches layouts.

🤖

AI Image Generation

Built-in OpenAI DALL-E 3 and Google Gemini Imagen support. Auto-generates images based on slide themes. Optional — works without API key.

✏️

Truly Editable

Real DrawingML shapes, not images. Every element can be clicked and edited in PowerPoint, Keynote, WPS, or Google Slides.

📏

8 Canvas Formats

16:9, 4:3, 9:16 portrait, 1:1 square, A4 portrait/landscape, LinkedIn banner, and any custom pixel dimensions.

🔒

100% Local & Private

All processing happens on your machine. No telemetry, no data collection. Only AI images call external APIs (can be disabled).

🆓

Free & Open Source

MIT License. Use freely for personal and commercial projects. No subscription, no hidden costs, no vendor lock-in.
